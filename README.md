# Personal Portfolio Website 

This is a responsive personal portfolio website built as part of my **Web Technology Lab Assignment 3**. It showcases my technical skills, projects, and educational background using modern web development practices.

 Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Styling, Animations, and Layouts.
* **Google Fonts:** Typography (Poppins).

This project implements the following technical requirements:
* **Flexbox Layout:** Used in the `Header` navigation and `Hero` section for alignment.
* **CSS Grid:** Used to organize the `Skills` and `Projects` sections into responsive cards.
* **Responsive Design:** Fully responsive layout that adapts to mobile, tablet, and desktop screens using Media Queries (`@media`).
* **Animations:**
    * Fade-in animation on page load.
    * Color-changing text effect on the "Full Stack Developer" title.
    * Hover effects on navigation links and project cards.
* **CSS Units:** Usage of relative units (`rem`, `%`) for scalability.
